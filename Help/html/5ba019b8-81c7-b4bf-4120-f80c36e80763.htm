      AppEnd Represents the AppEnd class that inherits from the base End class. This class manages application-specific behaviors related to restrictions.
      AppFor Represents the application-specific implementation of the "For" command.
      AppIf Represents the application-specific implementation of the "If" command.
      AppInt Represents the application-specific implementation of the "Int" command. This class manages an integer variable with restrictions on its value.
      AppMethod Represents the AppMethod class, which inherits from the Method base class. This class is designed to handle application-specific restriction logic by overriding methods in the base class.
      AppReal Represents the application-specific implementation of the "Real" command. This class manages a real (double) variable with restrictions on its value.
      AppRectangle Represents the application-specific implementation of the "Rect" command. This class handles the creation of a rectangle with an optional "filled" property.
      AppReset Represents the application-specific implementation of the "Reset" canvas command. This class handles resetting the canvas to its initial state.
      AppStoredProgram Represents an application-specific implementation of a stored program. This class manages the execution of commands in a stored program with an execution cycle limit and error handling.
      AppTriangle Represents an application-specific implementation of a triangle command. This class is responsible for drawing a triangle on the canvas with specified width and height.
      AppWhile Represents an application-specific implementation of a while loop command. This class is responsible for reducing restrictions related to the while loop.
      AppWrite Represents a command that writes text to the canvas. This command takes a single parameter (text) and writes it after evaluating any expressions in the text.
      Form1 Represents the main form of the application where the user interacts with the canvas and executes commands.
